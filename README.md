# **RefreshLayout**
基于google官方的下拉刷新，从新构建多功能刷新控件
## 使用方法

build.gradle文件中添加:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
并且:

```gradle
dependencies {
    implementation 'com.github.ashLikun:RefreshLayout:{latest version}'
}
```
